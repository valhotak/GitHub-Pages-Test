# Jak to funguje

GitHub Pages vezme obsah slozky `docs` a vystavi ho jako staticky web. Markdown soubory se prevedou na HTML, takze `docs/index.md` se chova jako hlavni stranka.

## Co je dulezite

- `docs/index.md` je domaci adresa webu.
- Odkazy mezi `.md` soubory zustanou prehledne i v repu.
- `docs/_config.yml` rika Pages webu jmeno, popis a tema.
- Neni potreba backend. Zadna databaze, zadne drama.

Kdyz udelas commit do publikovane branche, Pages se znovu sestavi. GitHub to obvykle zvladne sam, protoze ma na to tlacitko, roboty a sebevedomi.

Mrkni jeste na [caste prusvihy](caste-prusvihy.md), nebo se vrat na [start](index.md).
