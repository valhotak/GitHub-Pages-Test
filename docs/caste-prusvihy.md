# Caste prusvihy

## Stranka nejde otevrit

Zkontroluj, ze je Pages zapnute v **Settings -> Pages** a ze je vybrana spravna branch a slozka `/docs`.

## Vidim README misto webu

Pro hlavni stranku pouzij `docs/index.md`. README je skvele pro repo, ale web chce vlastni vchod.

## Odkazy nefunguji

Pouzivej relativni odkazy, treba `[Rychly start](rychly-start.md)`. Absolutni odkazy jsou fajn, dokud se nerozhodnou byt moc chytre.

## Zmeny se neprojevily

Pockej minutu nebo dve a obnov stranku. Pokud nic, podivej se na zalozku **Actions**, jestli build nespadl na necem konkretnejsim nez "vesmir dnes nespolupracuje".

Zpatky na [uvodni stranku](index.md).
