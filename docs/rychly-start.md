# Rychly start

Chces web hned. Rozumne. Tady je nejkratsi cesta:

1. Otevri repository na GitHubu.
2. Jdi do **Settings**.
3. V levem menu klikni na **Pages**.
4. U **Build and deployment** vyber **Deploy from a branch**.
5. Zvol branch, typicky `main`.
6. Jako slozku vyber `/docs`.
7. Klikni na **Save**.

Po chvili GitHub ukaze URL. Jestli ji neukaze hned, neni to chyba, jen mala internetova pauza na kavu.

Zpatky na [uvodni stranku](index.md), nebo dal na [jak to funguje](jak-to-funguje.md).
